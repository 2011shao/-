# 测试用例 v1.0

## 1. 测试范围

- 基础资料
- 入库管理
- 串码中心
- 库存与调拨盘点
- 权限与审计

## 2. 核心用例

| 编号 | 模块 | 用例标题 | 前置条件 | 预期结果 |
|---|---|---|---|---|
| TC-001 | 入库 | 手动录入串码成功 | 有品牌/分类/供应商 | 入库审核成功，库存+1，串码状态为在库 |
| TC-002 | 入库 | 重复串码拦截 | 串码已存在 | 返回业务冲突错误 |
| TC-003 | 入库权限 | 总店可选其他店铺入库 | 总店角色登录 | 可选择其他店铺 |
| TC-004 | 入库权限 | 分店不可选其他店铺 | 分店角色登录 | 仅可选择本店 |
| TC-005 | 自动串码 | 按数量自动生成串码 | 选择自动模式，qty=10 | 生成10个唯一串码 |
| TC-006 | 追溯 | 查询串码轨迹 | 串码已发生至少2个事件 | 轨迹完整且按时间排序 |
| TC-007 | 调拨 | 串码跨店调拨成功 | 源店有库存 | 源店减、目标店增、日志完整 |
| TC-008 | 并发 | 并发提交同串码 | 两请求同时提交 | 仅1个成功，另1个失败 |

## 3. 非功能测试

- 性能：关键列表接口 P95 < 500ms
- 稳定性：连续 2 小时回归链路无阻塞
- 安全：越权访问接口返回 403
