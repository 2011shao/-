2. 后端完成首版脚手架（Express + Prisma + PostgreSQL）
3. 前端完成首版 PC 管理台脚手架（Vue3 + setup + Pinia + Arco）
4. Docker Compose 部署骨架完成（db + erpserver + adminpc）
5. 第二阶段完成：
   - 后端补齐库存、调拨、盘点模块 API

   - 用户上下文与角色中间件（HQ/BRANCH/AUDITOR）

7. 第四阶段完成：
   - 调拨与盘点列表支持分页与状态筛选

8. 第五阶段完成：
   - 新增操作日志模型 `OperationLog`

9. 第六阶段完成：
   - 新增调拨单反审核接口（安全校验后回退库存位置）

   - 新增操作日志 CSV 导出接口
